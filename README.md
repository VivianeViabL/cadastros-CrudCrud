Site simples de cadastro de clientes desenvolvido com HTML5, CSS3 e JavaScript, utilizando a API externa "CrudCrud" para armazenamento temporário dos dados, 
e possui funcionalidades através dos métodos de GET (busca), POST (inserção) e DELETE (remoção).
Testes de requisição HTTP feitos pelo Postman.
*O CrudCrud possui limite de requisições e os dados expiram após um tempo em torno de 24 horas.
